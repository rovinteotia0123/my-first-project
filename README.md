# 🚀 Autonomous Cloud Cost Intelligence

## 📌 Overview
This project predicts future cloud costs using Machine Learning and provides insights through an interactive dashboard.

It helps organizations monitor, analyze, and optimize cloud expenses effectively.

---

## ✨ Features
- 🔮 Cloud cost prediction using ML (Linear Regression)
- ⚠️ Risk analysis (Stable / Moderate / High)
- 📊 Interactive dashboard with charts
- 🗃️ SQLite database for storing history
- 📜 History table + visualization
- 🔐 User login system
- 🎨 Light/Dark mode UI

---

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** HTML, CSS, JavaScript
- **ML Model:** Scikit-learn
- **Database:** SQLite
- **Visualization:** Chart.js
